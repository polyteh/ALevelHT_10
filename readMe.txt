Есть склад, где хранятся некоторые товары. Склад характеризуется объемом.
Каждое товар характеризуется: Название Цена Базовый товар - базовый товар несет в себе срок годности с момента поставки (по истечению - списание). Например, песок может храниться 5 дней, а проволока - только 3.
Каждый товар имеет метод эмулятор просрочки – по указанной дате поставки возвращает дату просрочки.
